# RGB Color Mixer Application

## Описание проекта

Простое WPF-приложение для смешивания цветов RGB с использованием трех ползунков. Приложение позволяет динамически изменять цвет фона путем регулировки значений красного, зеленого и синего каналов.

## Функциональность

- Три независимых ползунка для управления цветовыми каналами: Red, Green, Blue
- Динамическое обновление цвета фона в реальном времени
- Диапазон значений каждого канала от 0 до 255
- Начальный цвет - черный (0, 0, 0)

## Используемые технологии

- **.NET Framework 4.7.2**
- **WPF (Windows Presentation Foundation)**
- **XAML** для описания интерфейса
- **C#** для бизнес-логики

## Особенности реализации

1. **Привязка событий**: Каждый ползунок привязан к событию ValueChanged
2. **Реальное время**: Цвет обновляется мгновенно при перемещении любого ползунка
3. **Преобразование типов**: Значения double с ползунков преобразуются в byte для цветовой модели
4. **Создание кисти**: Динамическое создание SolidColorBrush на основе RGB значений

## Логика работы

```csharp
// При изменении значения любого ползунка:
// 1. Получаем текущие значения с трех ползунков
// 2. Преобразуем значения double в byte (0-255)
// 3. Создаем новый цвет с помощью Color.FromRgb()
// 4. Применяем цвет как фон для Grid
```
