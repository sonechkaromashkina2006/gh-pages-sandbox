# WPF Image Visibility Toggle Application

## Описание проекта

Простое WPF-приложение для отображения и скрытия изображения с помощью кнопки-переключателя.

## Функциональность

- Отображение изображения на главном окне
- Кнопка для переключения видимости изображения
- Автоматическое изменение текста кнопки в зависимости от состояния
- Приятный фиолетовый фон интерфейса

## Используемые технологии

- **.NET Framework 4.7.2**
- **WPF (Windows Presentation Foundation)**
- **XAML** для описания интерфейса
- **C#** для бизнес-логики

## Особенности реализации

1. **Toggle-кнопка**: Кнопка меняет свой текст и функциональность в зависимости от состояния
2. **Visibility управление**: Используются значения `Visibility.Visible` и `Visibility.Collapsed`
3. **Ресурсы изображений**: Изображения добавлены как ресурсы проекта
4. **Адаптивный дизайн**: Использование `Margin` и `Grid` для адаптивного размещения

## Логика работы

```csharp
// При нажатии кнопки:
// 1. Проверяем текущую видимость изображения
// 2. Если видно - скрываем и меняем текст кнопки на "show photo"
// 3. Если скрыто - показываем и меняем текст кнопки на "hide photo"
```
