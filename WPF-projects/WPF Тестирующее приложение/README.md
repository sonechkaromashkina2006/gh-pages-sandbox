# WPF Тестирующее приложение

## Описание проекта

Универсальное WPF-приложение для проведения тестов с поддержкой различных типов вопросов и загрузкой тестов из внешних файлов.

## Функциональность

- Загрузка тестов из текстовых файлов формата .txt
- 4 типа вопросов:
  1. Одиночный выбор (RadioButton)
  2. Множественный выбор (CheckBox)
  3. Текстовый ответ (TextBox)
  4. Вопрос с изображением (Image + RadioButton)
- Автоматическая проверка ответов
- Сброс всех ответов
- Адаптивный интерфейс с прокруткой

## Формат файла теста

Файл должен содержать данные в следующем порядке:

```
1. Текст первого вопроса (одиночный выбор)
Вариант ответа 1
Вариант ответа 2
Вариант ответа 3
Правильный ответ (точное совпадение)

[пустая строка]

2. Текст второго вопроса (множественный выбор)
Вариант 1
Вариант 2
Вариант 3
Вариант 4
Правильные ответы через запятую

[пустая строка]

3. Текст третьего вопроса (текстовый ответ)
Правильный ответ

[пустая строка]

4. Текст четвертого вопроса (с изображением)
Абсолютный путь к изображению
Вариант ответа 1
Вариант ответа 2
Вариант ответа 3
Правильный ответ
```

## Используемые технологии

- .NET Framework 4.7.2
- WPF (Windows Presentation Foundation)
- XAML для описания интерфейса
- C# для бизнес-логики

## Особенности реализации

1. **Динамическая загрузка**: Вопросы загружаются из внешнего файла при запуске
2. **Разные типы вопросов**: Поддержка 4 различных форматов вопросов
3. **Работа с изображениями**: Автоматическая загрузка и отображение картинок из указанных путей
4. **Проверка ответов**: Интеллектуальная проверка с учетом регистра и порядка ответов
5. **Гибкий интерфейс**: Использование ScrollViewer для удобного просмотра длинных тестов

## Логика работы

1. Пользователь нажимает "Загрузить вопросы из файла"
2. Выбирает текстовый файл с тестом
3. Приложение парсит файл и заполняет интерфейс вопросами
4. Пользователь отвечает на вопросы
5. Нажатие "Проверить ответы" вычисляет результат
6. "Сброс" очищает все ответы
