# Calculator — Калькулятор на Windows Forms (C++/CLI)

## Описание проекта

Графический калькулятор с пользовательским интерфейсом на Windows Forms, написанный на C++/CLI.

## Функциональность

Калькулятор поддерживает базовые математические операции:

1. **Арифметические операции:**
   * Сложение (+)
   * Вычитание (-)
   * Умножение (*)
   * Деление (/)
2. **Дополнительные функции:**
   * Изменение знака числа (+/-)
   * Добавление десятичной точки (.)
   * Вычисление квадратного корня (sqrt)
   * Очистка экрана (AC)
3. **Особенности интерфейса:**
   * Цветовое разделение цифровых и функциональных кнопок
   * Поддержка плавающей точки
   * Обработка деления на ноль
   * Кнопка закрытия приложения

## Особенности реализации

### Технологии:

* **C++/CLI** — расширение C++ для .NET Framework
* **Windows Forms** — графическая библиотека для интерфейса
* **.NET Framework 4.7.2** — целевая версия платформы

### Архитектура:

* **Однооконное приложение** без рамки (`FormBorderStyle::None`)
* **Обработчики событий** для каждой кнопки
* **Локальные переменные** для хранения состояния калькулятора
* **Валидация ввода** (проверка деления на ноль)
