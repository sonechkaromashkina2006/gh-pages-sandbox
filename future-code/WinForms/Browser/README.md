# Browser — Веб-браузер на Windows Forms (C++/CLI)

## Описание проекта

Простой веб-браузер с графическим интерфейсом, написанный на C++/CLI с использованием компонента WebBrowser.

## Функциональность

Браузер предоставляет базовые функции навигации:

1. **Навигация по URL:**

   - Ввод адреса в текстовое поле
   - Кнопка "Применить" для перехода по введенному URL
2. **Управление историей:**

   - Кнопка "<<" для перехода назад
   - Кнопка ">>" для перехода вперед
3. **Особенности интерфейса:**

   - Компонент WebBrowser для отображения веб-страниц
   - Автоматическое изменение заголовка окна
   - Окно разворачивается на весь экран

## Интерфейс

Браузер имеет простой интерфейс с элементами управления:

1. **Поле ввода URL** — для ввода веб-адресов
2. **Кнопка "Применить"** — переход по введенному URL
3. **Кнопки навигации** — "<<" (назад) и ">>" (вперед)
4. **Область просмотра** — компонент WebBrowser для отображения страниц

## Особенности реализации

### Технологии:

- **C++/CLI** — расширение C++ для .NET Framework
- **Windows Forms** — графическая библиотека для интерфейса
- **WebBrowser control** — встроенный компонент для веб-навигации
- **.NET Framework 4.7.2** — целевая версия платформы

### Архитектура:

- **Однооконное приложение** с поддержкой полноэкранного режима
- **Компонент WebBrowser** для рендеринга веб-страниц
- **Простая навигация** с сохранением текущего и предыдущего URL
- **Динамический заголовок** окна с отображением текущего сайта
