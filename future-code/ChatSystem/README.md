# ChatSystem - Простой клиент-серверный чат на C++

Простой многопользовательский чат на сокетах Windows с поддержкой многопоточности.

## Описание

### Server.cpp

- **Многопоточный сервер** на WinSock2 API
- Поддерживает **до 100 одновременных подключений**
- Каждое подключение обрабатывается в **отдельном потоке**
- **Широковещательная рассылка** - все сообщения отправляются всем клиентам
- Порт: **1111**, IP: **127.0.0.1** (localhost)

### Client.cpp

- **Клиент для подключения к серверу**
- **Двухпоточная архитектура**:
  - Главный поток: ввод и отправка сообщений
  - Второй поток: прием и вывод сообщений от сервера
- **Консольный интерфейс** для общения в реальном времени

## Быстрый старт

### Требования

- **Windows** (использует WinSock2 и WinAPI)
- Компилятор C++ (MinGW, MSVC)
- Отключенный брандмауэр для локального подключения

### Компиляция

**Компиляция сервера:**

```bash
g++ Server.cpp -o Server.exe -lws2_32 -mconsole
```

**Компиляция клиента:**

```bash
g++ Client.cpp -o Client.exe -lws2_32 -mconsole
```

### Запуск

1. **Запустите сервер:**

   ```
   Server.exe
   ```
   Вы увидите: `Client connected` при каждом новом подключении
2. **Запустите один или несколько клиентов:**

   ```
   Client.exe
   ```
   При успешном подключении: `Connected`
3. **Начните общение:**

   - Вводите сообщения в клиенте и нажимайте Enter
   - Сообщения будут видны всем подключенным клиентам
